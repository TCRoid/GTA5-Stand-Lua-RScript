# Stand-Lua-RScript

自己编写的自用脚本，把自己常用功能整合一起，部分代码借鉴参考（~~Copy~~）了 Heist Control、Wiri、Lance等等  
自己想要啥功能而且Stand没有或者突然有啥想法了，就会写一下脚本  
新手一个，代码很乱  



# 目前功能

### 自我选项

- 自定义最大生命值
- 自定义最大护甲值
- 自定义生命恢复
  - 站立、掩体内和载具内的恢复程度、恢复速度
- 自定义血量下限：到达设置的最低血量时
  - 锁定血量
  - 恢复血量
- 显示血量和护甲
- 警察无视
- 所有人无视
- 行动无声
- 不会被帮派骚乱
- 不可被拽出载具
- 载具内不可被射击
- 禁止为玩家调度警察
- 禁用NPC伤害

### 武器选项

- 可以射击队友
- 翻滚时自动换弹夹
- 无限弹药（不换弹）
- 锁定最大弹药
- 视野工具枪
  - 爆炸
  - 射击

- **实体控制枪**（控制瞄准的实体，并且会记录实体，可远程控制实体）
  - 描绘实体连线
  - 描绘实体边界框
  - 显示实体信息、复制实体信息
  - 保存到 Hash 列表
  - 请求控制实体
  - Entity :
    - Entity 选项
    - 传送 选项
    - 移动 选项
  - Ped :
    - Ped 选项
    - NPC作战能力选项
    - Ped Task 选项
  - Vehicle :
    - Vehicle 选项
    - Vehicle Task 选项

### 载具选项

- 载具改装
  - 自定义改装方案
  - 自动改装
  - 一键改装
- 载具车窗
  - 修复车窗
  - 摇下车窗
  - 粉碎车窗
- 载具门
  - 打开车门
  - 关闭车门
  - 破坏车门
  - 删除车门
  - 车门不可损坏
- 载具锁门
  - 选择锁门类型 设置载具门锁状态
  - 设置单个门锁状态（似乎无效）
  - 对所有玩家锁门
- 载具车灯
  - 设置车灯状态
  - 车灯不会损坏
- 载具电台
  - 选择电台 自动更改电台
  - 关闭电台
- 个人载具
  - 打开引擎和左车门
  - 开启载具引擎
  - 打开左车门
  - 打开左右车门
  - 传送到附近
- 载具武器
- 设置和锁定载具灰尘程度
- 防弹轮胎
- 载具引擎快速开启
- 解锁正在进入的载具
- 禁用载具喇叭

### 世界实体选项

- 管理附近的NPC、摄像头和门
  - NPC：移除武器、删除、死亡、爆头击杀
  - 摄像头：上下移动、删除
  - 门（本地有效）
    - 佩里科岛的门：删除、开关无碰撞
- 管理附近载具
  - 恶搞选项
  - 友好选项
  - 全部范围
    - 解锁车门
    - 打开左右车门和引擎
    - 拆下左右车门和打开引擎
- 管理附近NPC
  - 恶搞选项
  - 友好选项
  - 全部范围
    - 设置NPC作战能力
    - 取消NPC执行的任务
- 管理附近区域
  - 清理区域
  - 射击区域
- 管理任务实体（一些便于做任务的选项）
- 管理所有实体
- 实体信息枪（查看实体信息）

### 战局选项

- 阻挡区域

### 保镖选项

- 保镖NPC
- 保镖直升机

### 保护选项

- 移除爆炸
- 移除粒子效果
- 移除黏弹和感应地雷
- 移除防空区域

### 任务选项

- 分红编辑（除了公寓其余三大抢劫的分红编辑）
- 赌场抢劫
  - 第二面板 自定义可选工作
- 任务生命数修改
- 资产监视
- Local Editor（本地变量编辑）
- Global Editor（全局变量编辑）
- Stat Editor
  - 游玩时间
  - 日期
  - 属性技能
- 任务助手
  - 附近街道：在附近街道生成载具
  - 玩家助手
  - ... （一些便于做任务的选项）
- 跳过破解
- Voltage Hack

### 其它选项

- 零食护甲编辑
- 标记点选项：在标记点位置
  - 生成载具
  - 生成拾取物
  - 爆炸、RPG轰炸
- 请求服务
  - 移除CEO技能费用和冷却时间
  - 移除CEO载具请求费用
  - 请求弹道装甲和火神机枪
  - 重型装甲包裹 传送到我
  - 请求RC坦克
  - 请求RC匪徒
  - 无视犯罪
    - 锁定倒计时
    - 警察无视犯罪（莱斯特）
    - 贿赂当局 倒计时时间
    - 贿赂当局（CEO）
  - 幽灵组织
    - 倒计时时间
    - 幽灵组织（似乎无效）

- 聊天选项
  - 打字时禁用来电电话
  - 记录打字输入内容
- 自动跳到下一条对话
- 停止对话
- 移除悬赏
- 模拟鼠标左键点击（用于拿取目标财物时）

### 玩家选项

- 恶搞选项
  - 生成笼子
  - 生成小查攻击
  - 生成载具内敌人攻击
  - 传送所有行人、载具等到此玩家
  - 控制此玩家附近载具
    - 附近司机追赶此玩家
    - Task Vehicle Mission Target This Player（给载具设置个任务）
  - 控制此玩家附近行人
    - 附近行人逮捕此玩家
    - 附近行人敌对此玩家
  - 循环举报
- 友好选项
  - 生成医药包、护甲、零食、降落伞、呼吸器、火神机枪
  - 循环称赞
- 自定义Model生成实体
